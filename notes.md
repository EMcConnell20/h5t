## Key Input List ##

*State Selection*

- c => Open apply condition state
- d => Select target and open apply damage state

*Turn Control*

- a => Use target's action
- b => Use target's bonus action
- r => Use target's reaction
- n => Advance turn

*Interface Inputs*

- s => Toggle info block mode (stats | combat card)
- q => Close application
